#  Polling & Event Management System

A full-stack Polling Application where users can create events, add polls, invite participants, vote, and view live results.

---

##  Features

- User Authentication (JWT)
- Create Events
- Add Polls to Events
- Invite Participants via Email
- Email Notifications (Nodemailer)
- Vote & View Live Results
- Dashboard (Host & Invited Events)
- Secure APIs

---

## Tech Stack

### Frontend
- React (Vite)
- Material UI (MUI)
- Axios
- React Router

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Nodemailer (Gmail SMTP)
